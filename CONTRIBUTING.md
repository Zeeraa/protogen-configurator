# Contributing
We are happy to accept contributions to the project, but please keep the following in mind:

* Features should be optional and disabled by default.
* Features must never interfere with other features.
* Changes to features must always be backwards compatible.
* Changes to the editor should never cause users to lose their current config.
* Always test your code to make sure nothing is broken.
* Don't break support for low-end microcontrollers by adding too many features.

If your feature goes against these guidelines, please consider forking the project instead.

### Usage of generative AI
You are allowed to use generative AI to assist in writing code for this project, but please don't fully rely on it, as it may cause unintended behaviour.

Examples of what's allowed:
* Using AI to assist in writing code.
* Using AI for spell checking and proofreading text.
* Using AI to look for bugs.

Examples of what's not allowed:
* Fully relying on AI to write code without extensive testing.
* Using AI-generated images.
