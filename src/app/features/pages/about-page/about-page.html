Protogen Configurator

A tool for configuring LED matrixes for Protogens.

Getting started

Start by wiring up your LED matrix displays and inputs to your microcontroller.

Matrix pinout

Below is the pinout used by the default configuration. If you want to change your pins, you can do so on the Matrix Config page.

Matrix Pin Arduino Pin
CS 10
DIN 11
CLK 13

If your matrixes are rotated, you can adjust the rotation on the Matrix Config page as well.

Inputs

Any unused digital pin can be used as an input to trigger actions. Actions are configured on the Config page and allow you to assign buttons to different faces.

The mode of the input pin can be configured in the dropdown. The available modes are:

  • Normal: The input will require an external pull-up or pull-down resistor.
  • Pull-up (Recommended): The input will be internally connected to a HIGH state. Connect to ground to activate.
  • Pull-down: The input will be internally connected to a LOW state. Not all microcontrollers support this mode, so use with caution.

You can also check the "invert" option to invert the input state.

Changing the default matrix layout

The default config uses 2 matrixes for the eyes, 4 for the mouth, and 1 for the nose. If you use a different layout, you can change the matrix configuration on the Matrix Config page.

Creating faces

You can create and edit faces on the Faces page. To draw a symmetric face, make sure to check the "Symmetric drawing" option in the editor.

After creating a face, you can assign it as the default face on the Config page and also assign it to inputs to trigger it.

Uploading to Microcontroller

To export the configuration as code for your microcontroller, go to the Import/Export page. From there, click the "Export for Arduino" button to generate and download the code as an Arduino sketch file.

After downloading the .ino file, you can open it in the Arduino IDE and flash it to your microcontroller.
